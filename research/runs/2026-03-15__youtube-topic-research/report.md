## 1. 調査サマリー

- **人気テーマ**: 初心者解説・相場解説・銘柄解説・税金・取引所比較が安定需要。
- **タイトル**: 60文字以内・冒頭にフック・数字でCTR約36%向上（データ: 120,703本分析）。
- **サムネイル**: 金融系は成果・数字・顔で8〜15% CTRが可能（一般3〜4%比）。
- **型**: ロング＝「明確な主張＋緊張＋数字/括弧」、ショート＝「感情＋短さ＋絵文字1〜2個」。

## 3. タイトル案（フォーマット別）

### 3.1 ロング動画（10〜20分）

- **型**: 明確な主張 ＋ 緊張感 ＋ 数字 or 括弧  
- **文字数**: 50〜70文字（モバイルで切れないよう60文字前後推奨）  
- **避ける**: 疑問形だけのタイトル、絵文字多用  

| テーマ例 | タイトル案 |
|----------|------------|
| 税金 | 仮想通貨の税金でやるべき5つ［確定申告までに］ |
| 取引所 | 取引所おすすめ5選［手数料・セキュリティ比較 2026］ |
| 暴落 | 暴落時にやってはいけない3つのこと |
| 失敗談 | 假想通貨で300万溶かした理由（同じ失敗を防ぐ） |
| 積立 | 月1万円から始める仮想通貨積立［3年シミュレーション］ |

**データ根拠**: 120,703本分析で「数字入りタイトルはCTR約36%向上」「括弧で補足するとクリック増」[SubSub / jemsu等]。

### 3.2 ショート（60秒前後）

- **型**: 感情 or 驚き ＋ 短さ ＋ 絵文字1〜2個  
- **文字数**: 約40〜50文字  

| テーマ例 | タイトル案 |
|----------|------------|
| 相場 | ビットコイン急騰の理由、3行で解説⚡ |
| 注意 | このDM来たら詐欺です🛑 |
| 用語 | ステーキングって何？30秒で説明💰 |
| ニュース | 今週の仮想通貨ニュースまとめ🔥 |

**データ根拠**: 高パフォーマンスShortsの約57%が絵文字使用、低パフォーマンスは17% [SubSub]。

### 3.3 ライブ配信

- **型**: 権威性 or トピック ＋ 緊急性  
- **例**: LIVE：今週の相場解説＋質問に答えます / LIVE：重要ニュース解説［急騰の理由］

## 4. サムネイル構造（再生されやすい型）

### 4.1 金融・仮想通貨向けの目標

- **CTR目標**: 8〜15%（一般3〜4%比で約3.8倍収益ポテンシャル）  
- **表示**: モバイル前提（約70%）。実サイズ160〜180pxで読めること。

### 4.2 構成要素

| 要素 | 推奨 |
|------|------|
| **文字** | 2〜5語。フォント80〜200px、白/黄＋黒縁でコントラスト高く。 |
| **色** | ネイビー・黒・深緑・ゴールドで信頼感。差し色は黄・赤・オレンジ。 |
| **人物** | 表情が分かる顔をサムネの30〜40%程度。 |
| **数字・成果** | 「〜円」「〜%」「Before/After」を視覚化。 |
| **矢印・図** | 視線誘導・比較構造をはっきりさせる。 |

### 4.3 避けること

- 他チャンネルと同質のチャート・素材だけのサムネ  
- 文字が多すぎ・小さすぎでスマホで読めない  
- 複数の焦点で何が主題か分からない  
- テーマ説明だけで「視聴者の得」が伝わらない  

**効果例**: 「税金の節約方法」→「8,400ドル節約した方法」に変えてCTR約90%向上した事例あり（金融系チャンネル）[TubeVertex等]。

### 4.4 テンプレート例（文言）

- 「【◯◯円】＋ 短い結果テキスト」  
- 「3つの〇〇／5つの××」  
- 「Before → After」のビジュアル  
- 「LIVE」「NEW」「2026」で鮮度を出す

## 5. 再生されやすい型のまとめ

### 5.1 ロング動画

1. **数字型**: 「5つの〇〇」「3分で分かる〇〇」  
2. **How to型**: 「〇〇しない方法」「〇〇するための手順」  
3. **好奇心ギャップ型**: 「〇〇をやめたら再生伸びた（理由は動画で）」  
4. **括弧で補足型**: 「〇〇完全ガイド（初心者向け）」「〇〇［2026最新］」  

### 5.2 ショート

1. **感情＋短さ＋絵文字**: 「これマジ？😱」「〇〇がやばい🔥」  
2. **意外な組み合わせ**: 「〇〇と△△を同時にやった結果」  
3. **疑問で止める**: 「〇〇って本当？」（本文で答える）  

### 5.3 バズりやすい共通点（データベース）

- **短いタイトル**: 高パフォーマンスは約65〜70文字、低パフォーマンスは80文字超が多い。  
- **数字の使用**: ロングで32〜40%、配信で約45%が数字入りで高成績。  
- **疑問形**: トップ側では約6〜8%のみ。使うなら「驚き・価値・感情」が伴うものに限定。  
- **大文字**: 1〜3語だけ強調。全角大文字は避ける。  
- **フォーマット一致**: ロングは説明と信頼、ショートは感情とスピード、配信は「今」と権威。

## 6. 初心者向け vs 中級者向けの使い分け

|  | 初心者向け | 中級者向け |
|---|------------|------------|
| **テーマ** | 用語解説、取引所選び、税金、安全、積立 | 銘柄分析、板・チャート、ステーキング、海外取引所、シミュレーション |
| **タイトル** | 「〇〇とは」「初めての〇〇」「おすすめ5選」 | 「〇〇の選び方」「〇〇の意味」「実績ベース」 |
| **サムネ** | 安心感・ステップ感（番号、チェックリスト） | 数字・チャート・成果（％・金額） |
| **長さ** | 5〜10分が取りやすい | 10〜20分で深掘り可 |

## 8. 次のアクション提案

1. **テーマ**: 上記20テーマから、チャンネル方針に合わせて5〜10本をピックしてクオーター計画にする。  
2. **タイトル**: 1本につき2案作り、括弧・数字の有無でA/Bテスト（YouTubeのテスト機能 or サムネ差し替え）。  
3. **サムネ**: テンプレを2〜3パターン用意（「数字＋顔」「Before/After」「リスト型」）して回し使い。  
4. **計測**: ロングはCTR・視聴維持率、ショートは再生数・完了率を週次で確認し、型を更新する。
