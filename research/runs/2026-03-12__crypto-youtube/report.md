3. **視聴者満足度**（CTR x 視聴維持率）をアルゴリズムに評価させる

## 4. チャンネル登録を促す動画の要件定義

### 4.1 視聴者心理モデル

```
知る → 興味 → 信頼 → 登録（行動）
```

**心理学的原理**:
- **単純接触効果**: 繰り返し目にすることで親近感・信頼感が生まれる
- **返報性**: 無料で価値ある情報を得ると「登録」で返したくなる
- **損失回避**: 「登録しないと次の重要情報を見逃す」と感じさせる
- **社会的証明**: 登録者数・コメント数が多いと信頼される

**登録率の目安**:
| 登録率 | 評価 |
|--------|------|
| 0.3%未満 | 自然増レベル |
| 0.3〜0.5% | 成長中 |
| 0.5%以上 | 人気チャンネル |

### 4.2 動画構成の要件

#### A. オープニング（0〜15秒）- 最重要

| 要素 | 要件 | 理由 |
|------|------|------|
| フック | 視聴者の課題/疑問を言語化 | 7-10秒で離脱判断される |
| 権威性提示 | 実績・専門性を1文で | 信頼構築の起点 |
| 予告 | 「この動画で○○がわかります」 | 視聴継続の動機付け |

**例**: 「ビットコインが10万ドルを突破した今、あなたが絶対に知るべき3つのアルトコインがあります。元○○の私が、データで解説します」

#### B. 本編（15秒〜）

| 要素 | 要件 |
|------|------|
| パターンインタラプト | 20-30秒ごとに視覚・音楽・ペースを変化させる |
| データ根拠 | 主張にはチャート・数値・出典を必ず付ける |
| ストーリー構造 | 「問題提起→分析→解決策→次のアクション」 |
| 視聴者参加 | 「あなたはどう思いますか？コメントで教えてください」 |

#### C. クロージング

| 要素 | 要件 |
|------|------|
| CTA | 登録メリットを具体的に伝える（「毎週○曜に最新分析を配信」） |
| 次回予告 | 「次回は○○について深掘りします」 |
| コミュニティ誘導 | コメント・SNSへの誘導 |

### 4.3 動画フォーマット要件

| 項目 | 推奨値 | 根拠 |
|------|--------|------|
| 長尺動画の長さ | 7〜15分 | YouTube推奨・視聴維持と広告の最適バランス |
| ショート動画 | 15〜35秒 | Shorts最適長 |
| 投稿頻度（長尺） | 週2〜3本 | 日刊より質重視が2026年トレンド |
| 投稿頻度（Shorts） | 週3〜5本 | 成長チャンネルの70%がこの頻度 |
| 解像度 | 1080p以上 | 信頼性の視覚的担保 |
| 字幕 | 必須（自動+手動修正） | 音なし視聴者対応・SEO効果 |

### 4.4 サムネイル要件

| 項目 | 要件 | CTR効果 |
|------|------|---------|
| サイズ | 1280×720px（16:9） | 公式推奨 |
| テキスト文字数 | 12文字以内（日本語6〜8文字） | テキスト少=CTR高 |
| 色使い | 2〜3色、高コントラスト | 赤/青、黄/紫、緑/青が高CTR |
| 人物表情 | 驚き・確信の感情表現 | CTR最大+30% |
| 暗号通貨固有 | チャート（上昇）+ コイン画像 + 数字 | 金融系の信頼感 |
| 色の意味 | 緑+青 = お金 x 信頼 | 心理的効果 |
| NG | 文字詰め込み・暗い画像・情報過多 | CTR低下 |

**CTRベンチマーク**:
| CTR | 評価 |
|-----|------|
| 2-4% | 平均 |
| 5-7% | 標準的 |
| 8-12% | 良好 |
| 15%+ | 優秀 |

### 4.5 タイトル要件

| 要素 | 要件 | 例 |
|------|------|-----|
| 文字数 | 30〜40文字（日本語） | 長すぎると切れる |
| 数字 | 具体的な数字を含める | 「3つの理由」「100倍銘柄」 |
| 感情トリガー | 好奇心・緊急性・損失回避 | 「知らないと損する」「今すぐ」 |
| キーワード | 検索キーワードを冒頭に | 「ビットコイン」「仮想通貨」を前方に |
| サムネとの連携 | サムネとタイトルで完結する1つのメッセージ | 重複NG・補完関係 |

### 4.6 SEO・説明文要件

| 項目 | 要件 |
|------|------|
| 説明文冒頭2行 | 検索に表示される部分にキーワード+要約 |
| タグ | 「仮想通貨」「ビットコイン」「投資」等を10-15個 |
| ハッシュタグ | #仮想通貨 #ビットコイン #暗号資産 等（3個まで表示） |
| チャプター | タイムスタンプ付き（00:00形式） |
| カード・終了画面 | 関連動画への誘導を必ず設定 |

```
木: ショート x 2
金: データ分析・予測（長尺 10分）
土: ショート x 1（週まとめ）
日: （休み or ライブ配信）
```

→ 長尺3本 + ショート5本 = 週8本

## 6. 差別化・ポジショニング戦略

### 既存チャンネルの空白地帯（ブルーオーシャン）

| ポジション | 競合状況 | 攻め方 |
|-----------|---------|--------|
| **AI x 暗号通貨 特化** | ほぼ皆無 | AIエージェント・分散AI計算を専門解説 |
| **日本規制・税制 特化** | 少数 | 金商法移行・確定申告を深掘り |
| **RWA（実物資産トークン化）** | 新興 | 不動産・債券のトークン化を実例で |
| **データ可視化 特化** | 少数（海外にBenjamin Cowen） | オンチェーンデータをグラフで解説 |
| **DePIN（分散型物理インフラ）** | ほぼ皆無 | Helium/Render等の実機レビュー |

### 差別化の3原則

1. **ニッチ x 深さ**: 「暗号通貨全般」ではなく特定テーマで日本一になる
2. **データファースト**: 感覚論ではなくオンチェーンデータ・統計で語る
3. **透明性**: ポジション（保有銘柄）を公開し、利益相反を排除する

## 8. 視聴維持率を高める技術的要件

| テクニック | 実装方法 | 効果 |
|-----------|---------|------|
| フック | 冒頭7-10秒で最大の疑問を提示 | 離脱防止 |
| パターンインタラプト | 20-30秒ごとにカメラ角度・テロップ・BGM変化 | 飽き防止 |
| オープンループ | 「後半で驚きの結論をお見せします」 | 最後まで視聴 |
| B-roll挿入 | チャート画面・ニュース画像・アニメーション | 視覚的飽き防止 |
| テロップ | 重要ポイントを文字で強調 | 理解促進 |
| チャプター | 目次表示で見たい部分にジャンプ可能 | 満足度向上 |

## 10. Go/No-Go 意思決定

### 今始めるべき理由 TOP 3

1. **市場タイミング**: BTC回復 + アルトシーズン開始 = 情報需要が爆発的に増加中
2. **日本市場の空白**: AI x 暗号通貨、RWA、DePIN等の日本語専門チャンネルがほぼ存在しない
3. **YouTube Shorts**: ショート動画による新規リーチが過去最大の効果を発揮している時期

### 最初の1アクション

> **「2026年注目の暗号通貨テーマ5選」を長尺（10分）+ショート（30秒x3本）で制作・投稿する**
