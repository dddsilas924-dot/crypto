少額運用の注意事項
最小注文額に注意：MEXC先物では一部銘柄で最小注文額が5〜10 USDTです。ポジションサイズ15%の場合、300 USDTの15% = 45 USDTとなり問題ありませんが、ドローダウン後に資金が減少した場合は注文が通らなくなる可能性があります。

手数料比率が相対的に大きい：1トレードあたりの利益が小さいため、手数料（メイカー0.02% / テイカー0.06%）のインパクトが大口運用より大きくなります。可能な限り指値注文（メイカー）を利用してください。
