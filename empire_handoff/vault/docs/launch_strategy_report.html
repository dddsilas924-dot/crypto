緊急停止ルール

条件アクション
ポートフォリオ全体のドローダウンが -5% を超えた 新規エントリーを停止。既存ポジションは利確/損切まで保持。原因調査。
ポートフォリオ全体のドローダウンが -10% を超えた 全ポジション即時クローズ。全Bot を disabled に変更。バックテストとライブの乖離を徹底調査。
単一Botが 5連敗以上 該当Botのみ paper に戻す。他のBotは継続。
MEXC API エラーが 連続3回 全Bot を一時停止。API状態を確認後に再開。
