1. Bot-MeanRevert 完全解析

1-1. コンセプト（なぜ利益が出るのか）

核心原理: 平均回帰（Mean Reversion）
市場が楽観的なとき（Fear 50-80）、一部の銘柄は短期的に「上がりすぎ」の状態になる。
20日移動平均から15%以上乖離した銘柄は、統計的に平均に戻る傾向がある。
この「戻り」をショート（空売り）で利益にする。

市場心理の解説

  • Fear 50-80 = みんなが「まだ上がる」と楽観的に考えている
  • 一部の銘柄は群集心理で短期的に買われすぎる（FOMO: Fear Of Missing Out）
  • 20日移動平均から+15%以上乖離 = 統計的に「異常な上昇」
  • この超過上昇分が縮小する方向（=下落方向）にショートを入れる
具体例:
TAOが1週間で+30%上昇、20日移動平均は+5%地点にある。
→ MA20乖離 = +25%（閾値15%を大幅超過）
→ 「上がりすぎ」と判定してショート。平均方向に戻ればショートが利益になる。

2. Bot-WeakShort 完全解析

2-1. コンセプト

核心原理: 弱者淘汰（Weak Hands Elimination）
BTCが上昇している（+1%以上）のに、ついていけないアルトコインが存在する。
「強い相場で弱い銘柄」は、その後さらに売られる傾向がある。
この「取り残された弱者」をショートする。
MeanRevert vs WeakShort の違い
MeanRevertWeakShort
戦略逆張り（上がりすぎを売る）順張り（下がっている弱者を売る）
対象MA20から+15%以上上方乖離した銘柄BTCに対して3%以上下方乖離した銘柄
前提平均に戻る弱い銘柄はさらに弱くなる
BTC条件なしBTC +1%以上（強気必須）
同じショートでも、ロジックは真逆。

3. Bot-Sniper 完全解析

3-1. コンセプト

核心原理: 精密狙撃（Precision Sniping）
恐怖相場（Fear<30）でBTCが急落（-3%以上）しているとき、
出来高が爆発的に増加し（5倍以上）、かつBTCと低相関（<0.3）の銘柄を
ロングする。条件が4つ全て同時成立しないと発火しない。
月0-2回の超低頻度・超精密トレード。
他のBotとの決定的な違い:
  • 資金の 5%のみ 使用（他は15-30%）
  • レバレッジ 10x（超ハイリスク・ハイリターン）
  • 4条件全て同時成立が必要（他は2-3条件）
  • 負けても全資金の 0.5-2.5%程度 の損失で済む
