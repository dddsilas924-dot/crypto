6. 高頻度トレードの教訓

なぜ3/5のHF Botが赤字か

  1. 0.22% RTコストの壁: TP 2%に対してコスト比率11%。WR 55%以上が黒字の最低条件だが、1h足のノイズで達成困難
  2. 偽シグナルの多さ: 1h足レベルのRSI/BB/ブレイクは偽信号が多い。日足の方が信号品質が高い
  3. トレンドの短命さ: Momentumの3連続陽線はノイズである可能性が高い。真のトレンドは日足で検出した方が確実
  4. ペアトレードの限界: 片側近似バックテストの精度問題 + 1h足レベルの相関は不安定

HF-FRArbが唯一黒字の理由

  • 明確なエッジ: 過度なモメンタム(8h +3%以上) + RSI極端値は統計的に反転しやすい
  • 保有時間0.6h: 極短保有でリスク最小化。方向性リスクを最小限に抑制
  • WR 50.4%: TP/SL比率 (1.5:1) × WR 50.4% でギリギリ黒字。改善余地あり

改善提案

  • コスト削減: maker注文化 (0.00% fee) でRT 0.10%に半減 → 全Botの収益改善
  • FR実データ: 実際のFunding Rateデータ取得でHF-FRArbの精度向上
  • 銘柄フィルタ: 出来高上位30銘柄に限定してスプレッドコスト削減
  • 時間帯フィルタ: UTC 8-16時（アジア圏取引時間）のみでノイズ削減
