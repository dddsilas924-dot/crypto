# パターンD プレイブック

## 概要
パターンD（本質Alpha）局面での具体的なエントリー/エグジット手順を定義する。
BTC下落 + BTC.D低下 + Total MC横ばい = BTC売却→特定アルト先行買い。
5条件すべて満たす場合のみエントリーし、レバ2-3x、SL髭下で執行する。

## 定義・目的
ドミナンス・マトリクスの6パターンのうち、パターンD（BTC下落 + BTC.D低下 + Total MC横ばい）は本システムのコアとなる局面。このパターンでBTC売却資金が特定のアルトに流入する構造を捉え、先行買いを実行する。

## 判定ロジック
### エントリー条件（すべて満たすこと）
1. ドミナンス・マトリクスがパターンD
2. Alpha陽転: 対BTC騰落 > 0
3. 聖域: 10/10安値の上で反発
4. 鉄則適合: 200MA上 + 5MA上
5. Tier 2通過: VETO無し + LCEF低リスク + FR/OI健全

### エントリー実行
- レバレッジ: 2-3x
- SL: 聖域価格（10/10安値）の下
- ポジションサイズ: リスク管理ルールに従う（1銘柄10-15%、同時最大5-7銘柄）

### ターゲット銘柄例（2026-03-09時点）
- BIO（DeSci/10/10髭下）
- PI（イベント/Pi Day）
- UNI（DeFi/バーン加速）
- ONDO（RWA）
- LDO（P/S低位）

### エグジット条件
- **利確**: +10%以上で部分利確。パターンB移行時に残り全利確
- **損切り**: 聖域割れ（SLヒット）
- **強制決済**: パターンC移行で全ポジションクローズ
- **命綱発動**: 5MA割れで即全決済

### パターンD→他パターン移行時の対応
- **D→B**: アルト祭突入。利確の好機。保有継続も可
- **D→C**: 全面安。即全切り。例外なし
- **D→E**: BTC横ばい化。新規エントリー停止、既存は保有継続
- **D→A**: BTC独走。アルトから資金流出。利確検討
