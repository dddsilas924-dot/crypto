# L19: WCA（Whale Cluster Analysis / クジラ結束力）

## 概要
特定のクジラ群の同時エントリー形跡を検出し、大口の協調行動を識別する。
個々のクジラではなく、複数クジラの結束パターンを検出する。
実装難易度が非常に高く、初期実装では省略推奨。

## 定義・目的
個々のクジラの行動ではなく、複数のクジラが同時期に同じ銘柄を蓄積/売却するパターン（結束力）を検出する。クジラ群が結束して蓄積している銘柄は、大きな上昇の前兆である可能性がある。

## 判定ロジック
- **結束蓄積**: 複数の大口ウォレットが同時期に同銘柄を蓄積 → 強いポジティブシグナル
- **結束売却**: 複数の大口ウォレットが同時期に同銘柄を売却 → 強いネガティブシグナル
- **分散**: クジラ間で方向が分かれている → 中立

## データ要件
- **大口ウォレットの取引履歴**: Nansen / Arkham Intelligence / Whale Alert
- **クジラクラスター分析**: Dune Analyticsカスタムクエリ
- **更新頻度**: 日次
- **MEXC互換性**: 不可（オンチェーンデータ）

## データプロキシ
- 簡易代替: top holder数の変化率（CoinGecko等で間接取得）
- CCXT大口取引の同時性分析（同一時間帯に複数の大口買い）
- 共通Twitter/holderクラスター（手動分析）

## 実装ヒント
- 実装難易度が非常に高い。初期実装では省略推奨
- 将来的にNansen/Arkham APIを利用
- 簡易版: CCXT tradesから大口取引（上位1%サイズ）を抽出し、同一方向の同時性を計算
- Tier: 3
- 計算コスト: 重量（オンチェーン分析）

## 他ロジックとの関係
- **依存するロジック**: L15（WSR: クジラ保持率と補完関係）
- **連携するロジック**: L16（SSD: クジラ蓄積による供給ショック）
- **矛盾・注意点**: クジラの協調行動はマーケット操作の可能性もあり、盲信は危険

- **スクリーニング**: Tier 3コメンタリーの参考材料
- **エントリー**: クジラ結束蓄積 → 強い買い材料（ただしTier 3参考情報）
