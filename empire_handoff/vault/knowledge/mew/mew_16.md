# L16: サーキットブレーカー

## 概要
異常市場環境を検出し、全取引を自動停止するリスク管理ロジック。
フラッシュクラッシュ、相関崩壊、ポートフォリオ急損、API障害の4条件を監視。
停止後は一定時間経過で自動再開。

## 核心内容
- **フラッシュクラッシュ検出**: 1分間で5%変動 → 全停止（4h後に再開）
- **BTC-Alt相関崩壊**: 4h相関係数 < 0.5 → 全停止（2h後に再開）
- **ポートフォリオ損失**: 1h内の損失 > 10% → 全停止
- **API遅延**: レスポンス > 1秒 → 全停止

## トレードロジックへの変換
1. 1分足ティッカーを常時監視し、変動率を計算
2. BTC価格とアルトコイン価格の4h相関係数をローリング計算
3. ポートフォリオ時価総額を1hウィンドウで追跡
4. API呼び出しのレイテンシを測定
5. いずれかの条件がトリガーされたら全オープンオーダーをキャンセル、新規エントリー禁止
6. 設定された再開時間後に条件を再チェックし、正常なら再開

## 既存システムとの統合ポイント
- 全ロジック(L01-L21)の上位に位置するメタレイヤーとして機能
- L17(相関シフト)の相関計算と部分的に共有可能
- L14(ポジションサイジング)のリスク配分を一時的にゼロに強制

## 実装に必要なデータ・API
- CCXT経由のティッカーデータ（リアルタイム価格変動監視）
- BTC価格フィード（相関計算用）
- ポートフォリオ残高API
- APIレスポンスタイム計測
