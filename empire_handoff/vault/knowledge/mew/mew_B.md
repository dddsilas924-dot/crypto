# Deep Logic B: ステーブル流入速度(CVM)

## 概要
ステーブルコインの純流入速度を測定し、市場への買い上げ燃料の有無を判定。
24h流入量と7日間平均の比率で「テレポート準備完了」状態を検出。
L18(グローバル流動性)を補完する短期流動性指標。

## 核心内容
- **Sシグナル**: 24hステーブル純流入 > 7日間平均の1.5倍 = テレポート準備完了
- **Zシグナル（危険）**: 比率 < 1.0 = 買い上げ燃料なし → ロング封印
- **原理**: ステーブルコインの急速な流入は買い圧力の先行指標

## トレードロジックへの変換
1. 主要ステーブルコイン(USDT/USDC)のCEXへの24h純流入量を取得
2. 過去7日間の日次平均純流入量を計算
3. CVM比率 = 24h純流入 / 7d平均純流入
4. CVM > 1.5 → マーケットレジーム「買い燃料充填」→ ロング許可
5. CVM < 1.0 → マーケットレジーム「燃料枯渇」→ ロング封印

## 既存システムとの統合ポイント
- L18(グローバル流動性)のステーブルコイン分析を短期視点で補完
- マーケットレジーム判定の追加レイヤーとして機能
- L14(ポジションサイジング)のリスク配分調整に影響

## 実装に必要なデータ・API
- CEXステーブルコイン流入/流出データ（CryptoQuant、Glassnode等）
- ステーブルコイン時価総額変化データ（DefiLlama等）
- オンチェーンステーブルコイン送金データ
