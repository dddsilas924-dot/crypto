## 概要

LevBurnの発展形。FR偏り検知で方向を決定し、1秒足リアルタイムデータでエントリータイミングを特定する。

## コアロジック

1. **FR偏り = 方向の予測**（どちら側が焼かれるか）
   - LevBurnと同じFR偏り判定を使用
   - FR > 0 → ロングポジション過多 → SHORT方向
   - FR < 0 → ショートポジション過多 → LONG方向

2. **1秒足 = タイミングの特定**（いつ焼きが始まるか）
   - 価格急変: 1秒/5秒/10秒の変動率
   - 出来高急増: 直近10秒の出来高が60秒平均の3倍以上
   - 約定偏り: buy/sell比率で清算カスケードの初動を検知
   - 板の急変: bid/ask壁の消失で大口の仕掛けを検知

3. **スコアリング**
   - FR側: 最大50点（FR強度 + Regime連携）
   - リアルタイム側: 最大70点（価格急変 + 出来高 + 約定偏り + 板）
   - 合計: 最大100点（60点以上でシグナル発火）

## トレード特性

- 保有時間: 数秒〜30分（最大）
- TP: +0.5〜3%（バリアントによる）
- SL: -0.2〜1%（タイト）
- レバレッジ: 7〜20x
- ポジションサイズ: 資金の1〜3%（小さく、高頻度）
